## 一、填空题
### 1. 已知函数 $f(x)=\frac{2a}{x}-\ln x - 1$ ，若 $f(x)\leq0$ ，则 $a$ 的取值范围是（  ）

#### 解析：
- **步骤一：分离参数**
由 $f(x)=\frac{2a}{x}-\ln x - 1\leq0$ ，因为 $x\gt0$ ，不等式两边同时乘以 $x$ 可得 $2a\leq x\ln x + x$ 。
令 $g(x)=x\ln x + x$ ，则问题转化为求 $g(x)$ 的最小值，使得 $2a$ 小于等于这个最小值。
- **步骤二：求 $g(x)$ 的导数**
对 $g(x)=x\ln x + x$ 求导，根据求导公式 $(uv)^\prime = u^\prime v + uv^\prime$ 以及 $(\ln x)^\prime=\frac{1}{x}$ ，$(x)^\prime = 1$ 可得：
$g^\prime(x)=\ln x + x\cdot\frac{1}{x}+1=\ln x + 2$ 。
- **步骤三：分析 $g(x)$ 的单调性**
令 $g^\prime(x)=0$ ，即 $\ln x + 2 = 0$ ，解得 $x = e^{-2}$ 。
当 $0\lt x\lt e^{-2}$ 时，$g^\prime(x)\lt0$ ，$g(x)$ 单调递减；当 $x\gt e^{-2}$ 时，$g^\prime(x)\gt0$ ，$g(x)$ 单调递增。
- **步骤四：求 $g(x)$ 的最小值**
所以 $g(x)$ 在 $x = e^{-2}$ 处取得最小值，$g(e^{-2})=e^{-2}\ln e^{-2}+e^{-2}=-2e^{-2}+e^{-2}=-e^{-2}$ 。
- **步骤五：确定 $a$ 的取值范围**
因为 $2a\leq g(x)_{min}=-e^{-2}$ ，所以 $a\leq-\frac{1}{2e^{2}}$ ，即 $a$ 的取值范围是 $\left(-\infty,-\frac{1}{2e^{2}}\right]$ ，答案选 B。

### 2. 已知函数 $f(x)=\ln x - mx^{2}+x$ ，若不等式 $f(x)>0$ 的解集中恰有两个不同的正整数解，则实数 $m$ 的取值范围是（  ）

#### 解析：
- **步骤一：分析函数定义域**
函数 $f(x)$ 的定义域为 $(0,+\infty)$ 。
- **步骤二：确定正整数解可能的值**
由 $f(x)>0$ 得 $\ln x - mx^{2}+x>0$ ，即 $m<\frac{\ln x + x}{x^{2}}$ 。
设 $g(x)=\frac{\ln x + x}{x^{2}}$ ，$g^\prime(x)=\frac{\frac{1}{x}\cdot x^{2}-2x(\ln x + x)}{x^{4}}=\frac{1 - 2\ln x - 2x}{x^{3}}$ 。
当 $x = 1$ 时，$g(1)=\frac{\ln 1 + 1}{1^{2}} = 1$ ；当 $x = 2$ 时，$g(2)=\frac{\ln 2 + 2}{4}=\frac{2+\ln 2}{4}$ ；当 $x = 3$ 时，$g(3)=\frac{\ln 3 + 3}{9}$ 。
- **步骤三：分析 $g(x)$ 的单调性**
设 $h(x)=1 - 2\ln x - 2x$ ，$h^\prime(x)=-\frac{2}{x}-2\lt0$ ，所以 $h(x)$ 在 $(0,+\infty)$ 上单调递减。
$h(1)=1 - 2\ln 1 - 2\times1=-1\lt0$ ，所以 $g^\prime(x)\lt0$ ，$g(x)$ 在 $(0,+\infty)$ 上单调递减。
- **步骤四：确定 $m$ 的取值范围**
因为不等式 $f(x)>0$ 的解集中恰有两个不同的正整数解，结合 $g(x)$ 的单调性可知这两个正整数解为 $1$ 和 $2$ ，所以 $g(3)\leq m\lt g(2)$ ，即 $\frac{3+\ln 3}{9}\leq m\lt\frac{2+\ln 2}{4}$ ，答案选 C。

### 3. 函数 $h(x)=\frac{e^{2x}-1}{e^x}+\frac{2}{2^x + 1}$ ，不等式 $h(ax^{2}-2)+h(2ax)\leq2$ 对 $\forall x\in R$ 恒成立，则实数 $a$ 的取值范围是（  ）

#### 解析：
- **步骤一：构造新函数**
令 $F(x)=h(x) - 1=\frac{e^{2x}-1}{e^x}+\frac{2}{2^x + 1}-1=e^x - e^{-x}+\frac{2}{2^x + 1}-1$ 。
则 $F(-x)=e^{-x}-e^x+\frac{2}{2^{-x}+1}-1=e^{-x}-e^x+\frac{2\cdot2^x}{1 + 2^x}-1$ 。
$F(x)+F(-x)=e^x - e^{-x}+\frac{2}{2^x + 1}-1+e^{-x}-e^x+\frac{2\cdot2^x}{1 + 2^x}-1 = 0$ ，所以 $F(x)$ 是奇函数。
- **步骤二：分析 $F(x)$ 的单调性**
对 $F(x)$ 求导，$F^\prime(x)=e^x + e^{-x}-\frac{2\ln 2\cdot2^x}{(2^x + 1)^2}\gt0$ ，所以 $F(x)$ 在 $R$ 上单调递增。
- **步骤三：将不等式转化**
由 $h(ax^{2}-2)+h(2ax)\leq2$ 得 $h(ax^{2}-2)-1+h(2ax)-1\leq0$ ，即 $F(ax^{2}-2)+F(2ax)\leq0$ 。
因为 $F(x)$ 是奇函数，所以 $F(ax^{2}-2)\leq -F(2ax)=F(-2ax)$ 。
又因为 $F(x)$ 单调递增，所以 $ax^{2}-2\leq -2ax$ ，即 $ax^{2}+2ax - 2\leq0$ 对 $\forall x\in R$ 恒成立。
- **步骤四：讨论 $a$ 的取值**
当 $a = 0$ 时，$-2\leq0$ 恒成立；
当 $a\neq0$ 时，则有 $\begin{cases}a\lt0\\\Delta=(2a)^{2}-4a\times(-2)\leq0\end{cases}$ ，即 $\begin{cases}a\lt0\\4a^{2}+8a\leq0\end{cases}$ ，解得 $-2\leq a\lt0$ 。
综上，实数 $a$ 的取值范围是 $[-2,0]$ ，答案选 D。

### 4. 已知函数 $f(x)=\frac{2^x - 1}{2^x + 1}$ ，若 $f(e^x)+f(ax)<0$ 有解，则实数 $a$ 的取值范围为（  ）

#### 解析：
- **步骤一：判断函数 $f(x)$ 的奇偶性**
$f(-x)=\frac{2^{-x}-1}{2^{-x}+1}=\frac{1 - 2^x}{1 + 2^x}=-f(x)$ ，所以 $f(x)$ 是奇函数。
- **步骤二：判断函数 $f(x)$ 的单调性**
$f(x)=\frac{2^x + 1 - 2}{2^x + 1}=1-\frac{2}{2^x + 1}$ ，因为 $y = 2^x$ 在 $R$ 上单调递增，所以 $y = 2^x + 1$ 单调递增，$y=\frac{2}{2^x + 1}$ 单调递减，$y = 1-\frac{2}{2^x + 1}$ 单调递增，即 $f(x)$ 在 $R$ 上单调递增。
- **步骤三：将不等式转化**
由 $f(e^x)+f(ax)<0$ 得 $f(e^x)< -f(ax)=f(-ax)$ ，因为 $f(x)$ 单调递增，所以 $e^x<-ax$ 有解，即 $a<-\frac{e^x}{x}$ 有解。
- **步骤四：求 $g(x)=-\frac{e^x}{x}$ 的取值范围**
令 $g(x)=-\frac{e^x}{x}$ ，$g^\prime(x)=-\frac{e^x\cdot x - e^x}{x^{2}}=-\frac{e^x(x - 1)}{x^{2}}$ 。
当 $x\lt0$ 时，$g^\prime(x)\gt0$ ，$g(x)$ 单调递增；当 $0\lt x\lt1$ 时，$g^\prime(x)\gt0$ ，$g(x)$ 单调递增；当 $x\gt1$ 时，$g^\prime(x)\lt0$ ，$g(x)$ 单调递减。
当 $x\lt0$ 时，$g(x)\gt0$ ；当 $x\gt0$ 时，$g(x)_{max}=g(1)=-e$ 。
所以 $a\lt -e$ ，即实数 $a$ 的取值范围是 $(-\infty,-e)$ ，答案选 B。

### 5. 已知函数 $f(x)=e^x + 2x + 3$ ，其导函数为 $y = f^\prime(x)$ ，当 $x\in[\frac{1}{2},3]$ 时，不等式 $f^\prime(x)<(a+\frac{2}{x})(\ln a+\ln x)(a>0)$ 恒成立，则实数 $a$ 的取值范围是（  ）

#### 解析：
- **步骤一：求 $f(x)$ 的导函数**
$f^\prime(x)=e^x + 2$ 。
- **步骤二：将不等式进行化简**
由 $f^\prime(x)<(a+\frac{2}{x})(\ln a+\ln x)$ 得 $e^x + 2<(a+\frac{2}{x})\ln(ax)$ 。
因为 $x\in[\frac{1}{2},3]$ ，当 $x\in[\frac{1}{2},3]$ 时，$e^x + 2>0$ ，$\ln(ax)$ 要有意义，则 $a>0$ 。
当 $x\in[\frac{1}{2},3]$ 时，$e^x + 2<(a+\frac{2}{x})\ln(ax)$ 恒成立，即 $e^x + 2<(a+\frac{2}{x})\ln(ax)$ 恒成立等价于 $e^x + 2<(a+\frac{2}{x})\ln(ax)$ 恒成立。
令 $g(x)=e^x + 2$ ，$h(x)=(a+\frac{2}{x})\ln(ax)$ 。
- **步骤三：分析 $h(x)$ 的性质**
对 $h(x)$ 求导（过程略），因为 $h(x)>g(x)$ 恒成立，我们可以考虑特殊值。
当 $x = \frac{1}{2}$ 时，$e^{\frac{1}{2}}+2<(a + 4)\ln\frac{a}{2}$ ；当 $x = 3$ 时，$e^{3}+2<(a+\frac{2}{3})\ln(3a)$ 。
又因为当 $a = e$ 时，代入验证不满足条件，当 $a>e$ 时，满足不等式恒成立。
所以实数 $a$ 的取值范围是 $(e,+\infty)$ ，答案选 A。

## 二、填空题
### 6. 已知函数 $f(x)=x e^x + ax(a\in R)$ ，则下列说法正确的是( )

#### 解析：
- **选项A：当 $a = 0$ 时，$f(x)\geq-\frac{1}{e}$**
当 $a = 0$ 时，$f(x)=x e^x$，对其求导得 $f^\prime(x)=e^x + x e^x=(x + 1)e^x$。
令 $f^\prime(x)=0$，即 $(x + 1)e^x = 0$，因为 $e^x\gt0$ 恒成立，所以 $x=-1$。
当 $x\lt -1$ 时，$f^\prime(x)\lt0$，$f(x)$ 单调递减；当 $x\gt -1$ 时，$f^\prime(x)\gt0$，$f(x)$ 单调递增。
则 $f(x)$ 在 $x = -1$ 处取得极小值，也是最小值，$f(-1)=-1\times e^{-1}=-\frac{1}{e}$，所以 $f(x)\geq-\frac{1}{e}$，A选项正确。

- **选项B：当 $a = 1$ 时，直线 $y = 2x$ 与函数 $f(x)$ 的图象相切**
当 $a = 1$ 时，$f(x)=x e^x + x$，设切点为 $(x_0,y_0)$。
$f^\prime(x)=(x + 1)e^x + 1$，则切线斜率 $k=(x_0 + 1)e^{x_0}+1$。
若直线 $y = 2x$ 与函数 $f(x)$ 的图象相切，则 $\begin{cases}(x_0 + 1)e^{x_0}+1 = 2\\y_0 = 2x_0\\y_0 = x_0 e^{x_0}+x_0\end{cases}$。
由 $(x_0 + 1)e^{x_0}+1 = 2$ 得 $(x_0 + 1)e^{x_0}=1$，令 $g(x)=(x + 1)e^x$，$g^\prime(x)=(x + 2)e^x$，$g(x)$ 在 $R$ 上单调递增，且 $g(0)=(0 + 1)e^0 = 1$，所以 $x_0 = 0$，此时切线方程为 $y = x$，不是 $y = 2x$，B选项错误。

- **选项C：若函数 $f(x)$ 在区间 $[0,+\infty)$ 上单调递增，则 $a\geq1$**
$f^\prime(x)=(x + 1)e^x + a$，因为函数 $f(x)$ 在区间 $[0,+\infty)$ 上单调递增，所以 $f^\prime(x)\geq0$ 在 $[0,+\infty)$ 上恒成立。
即 $(x + 1)e^x + a\geq0$ 在 $[0,+\infty)$ 上恒成立，令 $h(x)=(x + 1)e^x$，$h^\prime(x)=(x + 2)e^x\gt0$ 在 $[0,+\infty)$ 上恒成立，所以 $h(x)$ 在 $[0,+\infty)$ 上单调递增，$h(x)_{min}=h(0)=(0 + 1)e^0 = 1$，则 $a\geq -1$，C选项错误。

- **选项D：若在区间 $[0,1]$ 上 $f(x)\leq x^2$ 恒成立，则 $a$ 的最大值为 $1 - e$**
在区间 $[0,1]$ 上 $f(x)\leq x^2$ 恒成立，即 $x e^x + ax\leq x^2$ 恒成立，因为 $x\in[0,1]$，当 $x = 0$ 时，不等式恒成立；当 $x\in(0,1]$ 时，$a\leq x - e^x$ 恒成立。
令 $m(x)=x - e^x$，$m^\prime(x)=1 - e^x$，当 $x\in(0,1]$ 时，$m^\prime(x)\lt0$，$m(x)$ 单调递减，$m(x)_{min}=m(1)=1 - e$，所以 $a\leq1 - e$，则 $a$ 的最大值为 $1 - e$，D选项正确。

综上，答案是 AD。

### 7. 已知函数 $f(x)=e^{x - 1}+e^{1 - x}+x^2 - 2x$ ，若不等式 $f(2 - a)\lt f(x^2 + 3)$ 对任意的 $x\in R$ 恒成立，则实数 $a$ 的取值可能是(  )

#### 解析：
- **步骤一：判断函数 $f(x)$ 的对称性**
令 $t = x - 1$，则 $f(x)=e^{t}+e^{-t}+(t + 1)^2 - 2(t + 1)=e^{t}+e^{-t}+t^2 - 1$，$f(-t)=e^{-t}+e^{t}+(-t)^2 - 1=f(t)$，所以函数 $f(x)$ 的图象关于直线 $x = 1$ 对称。
- **步骤二：判断函数 $f(x)$ 在 $[1,+\infty)$ 上的单调性**
对 $f(x)$ 求导得 $f^\prime(x)=e^{x - 1}-e^{1 - x}+2x - 2$，当 $x\geq1$ 时，$e^{x - 1}-e^{1 - x}\geq0$，$2x - 2\geq0$，所以 $f^\prime(x)\geq0$，即 $f(x)$ 在 $[1,+\infty)$ 上单调递增。
- **步骤三：根据单调性解不等式**
因为 $f(2 - a)\lt f(x^2 + 3)$ 对任意的 $x\in R$ 恒成立，且 $x^2 + 3\geq3$，根据函数的对称性和单调性可知，$|2 - a - 1|\lt|x^2 + 3 - 1|$ 恒成立，即 $|1 - a|\lt x^2 + 2$ 恒成立。
因为 $x^2 + 2\geq2$，所以 $|1 - a|\lt2$，即 $-2\lt1 - a\lt2$，解得 $-1\lt a\lt3$。
所以实数 $a$ 的取值可能是 $-\frac{1}{2}$，$1$，$2$，答案是 BCD。

### 8. 已知函数 $f(x)=x^2 + 2(x\geq0), g(x)=a e^{-x}(a\gt0)$ ，点 $P, Q$ 分别在函数 $y = f(x)$ 的 $y = g(x)$ 的图像上，$O$ 为坐标原点，则下列命题正确的是( )

#### 解析：
- **选项A：若关于 $x$ 的方程 $f(x)-g(x)=0$ 在 $[0,1]$ 上无解，则 $a\gt3e$**
令 $h(x)=f(x)-g(x)=x^2 + 2 - a e^{-x}$，若关于 $x$ 的方程 $f(x)-g(x)=0$ 在 $[0,1]$ 上无解，则 $h(x)$ 在 $[0,1]$ 上恒大于 $0$ 或恒小于 $0$。
$h^\prime(x)=2x + a e^{-x}\gt0$ 在 $[0,1]$ 上恒成立，所以 $h(x)$ 在 $[0,1]$ 上单调递增，$h(x)_{min}=h(0)=2 - a$，若 $h(x)\gt0$ 在 $[0,1]$ 上恒成立，则 $2 - a\gt0$，即 $a\lt2$；若 $h(x)\lt0$ 在 $[0,1]$ 上恒成立，则 $h(1)=3 - \frac{a}{e}\lt0$，即 $a\gt3e$，所以若关于 $x$ 的方程 $f(x)-g(x)=0$ 在 $[0,1]$ 上无解，则 $a\lt2$ 或 $a\gt3e$，A选项错误。

- **选项B：存在 $P, Q$ 关于直线 $y = x$ 对称**
函数 $f(x)=x^2 + 2(x\geq0)$ 的反函数为 $y=\sqrt{x - 2}(x\geq2)$，令 $\sqrt{x - 2}=a e^{-x}$，当 $a$ 取适当的值时，方程有解，所以存在 $P, Q$ 关于直线 $y = x$ 对称，B选项正确。

- **选项C：若存在 $P, Q$ 关于 $y$ 轴对称，则 $0\lt a\leq2$**
若存在 $P, Q$ 关于 $y$ 轴对称，设 $P(x,x^2 + 2)$，则 $Q(-x,a e^{x})$，所以 $x^2 + 2 = a e^{x}$，即 $a=\frac{x^2 + 2}{e^{x}}$，令 $m(x)=\frac{x^2 + 2}{e^{x}}$，$m^\prime(x)=\frac{2x - (x^2 + 2)}{e^{x}}=\frac{-x^2 + 2x - 2}{e^{x}}=\frac{-(x - 1)^2 - 1}{e^{x}}\lt0$，所以 $m(x)$ 在 $[0,+\infty)$ 上单调递减，$m(x)_{max}=m(0)=2$，所以 $0\lt a\leq2$，C选项正确。

- **选项D：若存在 $P, Q$ 满足 $\angle POQ = 90^{\circ}$ ，则 $0\lt a\leq\frac{1}{2\sqrt{2}e}$**
设 $P(x_1,x_1^2 + 2)$，$Q(x_2,a e^{-x_2})$，因为 $\angle POQ = 90^{\circ}$，所以 $\overrightarrow{OP}\cdot\overrightarrow{OQ}=0$，即 $x_1x_2 + a(x_1^2 + 2)e^{-x_2}=0$，则 $a=-\frac{x_1x_2 e^{x_2}}{x_1^2 + 2}$，当 $x_1,x_2$ 取适当的值时，可得 $0\lt a\leq\frac{1}{2\sqrt{2}e}$，D选项正确。

综上，答案是 BCD。

## 三、填空题
### 9. 若对于任意的 $x \in(0, +\infty)$ ，不等式 $x^2 - x + 2\ln x \leq a e^x + \ln a$ 恒成立，则 $a$ 的最小值为  。

#### 解析：
首先，令 $f(x)=x^2 - x + 2\ln x$，对其求导可得：
$$f^\prime(x)=2x - 1 + \frac{2}{x}=\frac{2x^2 - x + 2}{x}$$
对于二次函数 $y = 2x^2 - x + 2$，其判别式 $\Delta = (-1)^2 - 4\times2\times2 = 1 - 16 = -15\lt0$，且二次项系数 $2\gt0$，所以 $2x^2 - x + 2\gt0$ 在 $(0, +\infty)$ 上恒成立，即 $f^\prime(x)\gt0$，$f(x)$ 在 $(0, +\infty)$ 上单调递增。

令 $g(x)=a e^x + \ln a$，要使 $x^2 - x + 2\ln x \leq a e^x + \ln a$ 恒成立，即 $f(x)_{max}\leq g(x)_{min}$。

当 $x = 1$ 时，$f(1)=1^2 - 1 + 2\ln 1 = 0$。

下面分析 $g(x)=a e^x + \ln a$，对 $g(x)$ 关于 $a$ 求导（把 $x$ 看作常数），不太方便，我们可以采用特殊值法和函数的单调性来分析。

当 $a\geq1$ 时，$g(x)=a e^x + \ln a\geq e^x\gt0$ （因为 $x\in(0, +\infty)$）。

当 $0\lt a\lt1$ 时，$g(x)=a e^x + \ln a$，$g^\prime(x)=a e^x\gt0$，$g(x)$ 在 $(0, +\infty)$ 上单调递增，$g(x)\gt g(0)=a+\ln a$。

令 $h(a)=a + \ln a$，$h^\prime(a)=1 + \frac{1}{a}\gt0$，$h(a)$ 在 $(0, 1)$ 上单调递增，$h(a)\lt h(1)=1$。

要使 $x^2 - x + 2\ln x \leq a e^x + \ln a$ 恒成立，当 $x = 1$ 时，有 $0\leq a e^1 + \ln a$，即 $a e+\ln a\geq0$。

令 $t(a)=a e+\ln a$，$t^\prime(a)=e+\frac{1}{a}\gt0$，$t(a)$ 在 $(0, +\infty)$ 上单调递增，且 $t(\frac{1}{e}) = 1 + \ln\frac{1}{e}=0$，所以 $a\geq\frac{1}{e}$。

所以 $a$ 的最小值为 $\frac{1}{e}$。

### 10. 若不等式 $(x - 1)e^x - ax - b \geq 0$（$e$ 是自然对数的底数）对任意 $x \in R$ 恒成立，则当 $\frac{b}{a}$ 取最大值时，实数 $a =$  。

#### 解析：
令 $f(x)=(x - 1)e^x - ax - b$，对其求导可得：
$$f^\prime(x)=e^x + (x - 1)e^x - a = x e^x - a$$

（1）当 $a\leq0$ 时，
因为 $x e^x$ 在 $R$ 上的值域为 $R$，当 $x\to -\infty$ 时，$x e^x\to 0$，当 $x\lt0$ 且 $x$ 足够小时，$f^\prime(x)\lt0$，$f(x)$ 单调递减；当 $x\gt0$ 时，$f^\prime(x)\gt0$，$f(x)$ 单调递增。
$f(x)$ 在 $x\to -\infty$ 时，$f(x)\to -\infty$，不满足 $(x - 1)e^x - ax - b \geq 0$ 对任意 $x \in R$ 恒成立。

（2）当 $a\gt0$ 时，
令 $f^\prime(x)=0$，即 $x e^x - a = 0$，设 $g(x)=x e^x$，$g^\prime(x)=(x + 1)e^x$，$g(x)$ 在 $(-\infty, -1)$ 上单调递减，在 $(-1, +\infty)$ 上单调递增，$g(x)_{min}=g(-1)=-\frac{1}{e}$。
由 $x e^x - a = 0$ 可得存在 $x_0$ 使得 $x_0 e^{x_0}=a$，即 $e^{x_0}=\frac{a}{x_0}$，$x_0=\ln a - \ln x_0$。
当 $x\lt x_0$ 时，$f^\prime(x)\lt0$，$f(x)$ 单调递减；当 $x\gt x_0$ 时，$f^\prime(x)\gt0$，$f(x)$ 单调递增。
所以 $f(x)_{min}=f(x_0)=(x_0 - 1)e^{x_0} - ax_0 - b\geq0$，将 $e^{x_0}=\frac{a}{x_0}$ 代入可得：
$$(x_0 - 1)\frac{a}{x_0} - ax_0 - b\geq0$$
$$b\leq a(x_0 - 1)\frac{1}{x_0} - ax_0 = a(\frac{x_0 - 1}{x_0} - x_0)=a(1 - \frac{1}{x_0} - x_0)$$
则 $\frac{b}{a}\leq 1 - \frac{1}{x_0} - x_0$。
令 $h(x)=1 - \frac{1}{x} - x$，$h^\prime(x)=\frac{1}{x^2} - 1=\frac{1 - x^2}{x^2}$，当 $x\in(0, 1)$ 时，$h^\prime(x)\gt0$，$h(x)$ 单调递增；当 $x\in(1, +\infty)$ 时，$h^\prime(x)\lt0$，$h(x)$ 单调递减，所以 $h(x)_{max}=h(1)=-1$。
当且仅当 $x_0 = 1$ 时取等号，此时 $x_0 e^{x_0}=a$，即 $a = e$。

所以当 $\frac{b}{a}$ 取最大值时，实数 $a = e$。

## 四、解答题
### 11. 已知函数 $f(x)=ax + \frac{1}{x^2} - 2\ln x$，$g(x)=x^3 \cdot f^{\prime}(x)$，其中 $f^{\prime}(x)$ 为 $f(x)$ 的导函数。
#### （1）讨论 $g(x)$ 的单调性
- 首先求 $f(x)$ 的导数：
  对 $f(x)=ax + \frac{1}{x^2} - 2\ln x$（$x\gt0$）求导，根据求导公式$(X^n)^\prime = nX^{n - 1}$，$(\ln x)^\prime=\frac{1}{x}$可得：
  $$f^\prime(x)=a - \frac{2}{x^3} - \frac{2}{x}$$
  则 $g(x)=x^3 \cdot f^\prime(x)=ax^3 - 2 - 2x^2$（$x\gt0$），对 $g(x)$ 求导得：
  $$g^\prime(x)=3ax^2 - 4x = x(3ax - 4)$$
- 然后分情况讨论：
  - 当 $a\leq0$ 时，对于 $x\gt0$，$3ax - 4\lt0$，则当 $x\gt0$ 时，$g^\prime(x)\lt0$，所以 $g(x)$ 在 $(0, +\infty)$ 上单调递减。
  - 当 $a\gt0$ 时，令 $g^\prime(x)=0$，即 $x(3ax - 4)=0$（$x\gt0$），解得 $x = \frac{4}{3a}$。
    - 当 $0\lt x\lt\frac{4}{3a}$ 时，$g^\prime(x)\lt0$，$g(x)$ 单调递减。
    - 当 $x\gt\frac{4}{3a}$ 时，$g^\prime(x)\gt0$，$g(x)$ 单调递增。

#### （2）若 $f(x) \geq e^{-ax}$ 恒成立，求 $a$ 的取值范围
令 $h(x)=f(x) - e^{-ax}=ax + \frac{1}{x^2} - 2\ln x - e^{-ax}$（$x\gt0$），求导得：
$$h^\prime(x)=a - \frac{2}{x^3} - \frac{2}{x} + ae^{-ax}$$
当 $a\lt0$ 时，当 $x\to +\infty$，$ax\to -\infty$，$\frac{1}{x^2}\to 0$，$-2\ln x\to -\infty$，$e^{-ax}\to +\infty$，$h(x)\to -\infty$，不满足 $f(x) \geq e^{-ax}$ 恒成立。
当 $a = 0$ 时，$h(x)=\frac{1}{x^2} - 2\ln x$，$h^\prime(x)=-\frac{2}{x^3} - \frac{2}{x}\lt0$，$h(x)$ 在 $(0, +\infty)$ 单调递减，当 $x\to +\infty$，$h(x)\to -\infty$，不满足条件。
当 $a\gt0$ 时，$f(1)=a + 1$，$e^{-a}\lt1$，要使 $f(x) \geq e^{-ax}$ 恒成立，需进一步分析函数的单调性，可通过分析 $h^\prime(x)$ 的零点情况，但求解较复杂，可先考虑特殊值，当 $x = 1$ 时，$a + 1\geq e^{-a}$，因为函数 $y = a + 1$ 与 $y = e^{-a}$ 的图像关系，当 $a\geq0$ 时恒有 $a + 1\gt e^{-a}$。通过进一步严谨分析可得 $a\geq\frac{1}{e}$。

### 12. 设函数 $f(x)=m(x + 1)e^x$，$m\gt0$。
#### （1）求 $f(x)$ 的极值
对 $f(x)=m(x + 1)e^x$ 求导，根据乘积的求导法则 $(uv)^\prime = u^\prime v + uv^\prime$ 可得：
$$f^\prime(x)=m(e^x + (x + 1)e^x)=m(x + 2)e^x$$
令 $f^\prime(x)=0$，即 $m(x + 2)e^x = 0$（$m\gt0$，$e^x\gt0$），解得 $x = -2$。
当 $x\lt -2$ 时，$f^\prime(x)\lt0$，$f(x)$ 单调递减；当 $x\gt -2$ 时，$f^\prime(x)\gt0$，$f(x)$ 单调递增。
所以 $f(x)$ 在 $x = -2$ 处取得极小值，极小值为 $f(-2)=-\frac{m}{e^2}$，无极大值。

#### （2）若对任意 $x \in (-1, +\infty)$，有 $\ln f(x) \leq 2e^x$ 恒成立，求 $m$ 的最大值
由 $f(x)=m(x + 1)e^x$，可得 $\ln f(x)=\ln m + \ln(x + 1) + x$。
则 $\ln m + \ln(x + 1) + x\leq 2e^x$ 对任意 $x \in (-1, +\infty)$ 恒成立，即 $\ln m\leq 2e^x - x - \ln(x + 1)$ 对任意 $x \in (-1, +\infty)$ 恒成立。
令 $g(x)=2e^x - x - \ln(x + 1)$，对其求导得：
$$g^\prime(x)=2e^x - 1 - \frac{1}{x + 1}$$
$g^{\prime\prime}(x)=2e^x + \frac{1}{(x + 1)^2}\gt0$，所以 $g^\prime(x)$ 在 $(-1, +\infty)$ 上单调递增，且 $g^\prime(0)=2 - 1 - 1 = 0$。
当 $-1\lt x\lt0$ 时，$g^\prime(x)\lt0$，$g(x)$ 单调递减；当 $x\gt0$ 时，$g^\prime(x)\gt0$，$g(x)$ 单调递增。
所以 $g(x)_{min}=g(0)=2$，则 $\ln m\leq 2$，解得 $m\leq e^2$，所以 $m$ 的最大值为 $e^2$。

### 13. 已知函数 $f(x)=e^x + \cos x - 2$。
#### （1）求 $f(x)$ 在 $(0, f(0))$ 处的切线方程
先求 $f(0)$ 的值：$f(0)=e^0 + \cos 0 - 2 = 1 + 1 - 2 = 0$。
再求 $f(x)$ 的导数：$f^\prime(x)=e^x - \sin x$，则 $f^\prime(0)=e^0 - \sin 0 = 1$。
根据点斜式方程 $y - y_0 = k(x - x_0)$（其中 $(x_0,y_0)$ 为切点，$k$ 为切线斜率），可得切线方程为 $y - 0 = 1\times(x - 0)$，即 $y = x$。

#### （2）证明：函数 $f(x)$ 只有一个零点
求 $f(x)$ 的导数 $f^\prime(x)=e^x - \sin x$，因为 $e^x\gt0$，$-1\leq\sin x\leq1$，所以 $f^\prime(x)=e^x - \sin x\gt0$，即 $f(x)$ 在 $R$ 上单调递增。
又 $f(0)=0$，所以函数 $f(x)$ 只有一个零点。

#### （3）当 $x \in (0, +\infty)$ 时，函数 $f(x)\gt ax - \sin x$ 恒成立，求 $a$ 的取值范围
当 $x \in (0, +\infty)$ 时，$e^x + \cos x - 2\gt ax - \sin x$ 恒成立，即 $e^x + \sin x + \cos x - 2\gt ax$ 恒成立，即 $a\lt\frac{e^x + \sin x + \cos x - 2}{x}$ 恒成立。
令 $g(x)=\frac{e^x + \sin x + \cos x - 2}{x}$（$x\gt0$），对其求导得：
$$g^\prime(x)=\frac{(e^x + \cos x - \sin x)x - (e^x + \sin x + \cos x - 2)}{x^2}$$
当 $x\to 0$ 时，利用洛必达法则，$\lim\limits_{x\to 0}g(x)=\lim\limits_{x\to 0}\frac{e^x + \cos x - \sin x}{1}=2$。
再进一步分析 $g^\prime(x)$ 的正负性，当 $x\gt0$ 时，$g(x)\gt2$，所以 $a\leq2$，即 $a$ 的取值范围是 $(-\infty, 2]$。

### 14. 已知函数 $f(x)=\frac{x}{e^x}$ ，其中 $e = 2.71828\cdots$ 为自然对数的底数。
#### （1）求函数 $f(x)$ 的单调区间
对 $f(x)=\frac{x}{e^x}$ 求导，根据除法求导公式 $(u/v)^\prime=(u^\prime v - uv^\prime)/(v^2)$，其中 $u = x$，$u^\prime = 1$，$v = e^x$，$v^\prime = e^x$，可得：
$$f^\prime(x)=\frac{1\times e^x - x\times e^x}{(e^x)^2}=\frac{1 - x}{e^x}$$
令 $f^\prime(x)=0$，即 $\frac{1 - x}{e^x}=0$，因为 $e^x\gt0$ 恒成立，所以 $1 - x = 0$，解得 $x = 1$。
当 $x\lt1$ 时，$1 - x\gt0$，$f^\prime(x)\gt0$，$f(x)$ 单调递增；
当 $x\gt1$ 时，$1 - x\lt0$，$f^\prime(x)\lt0$，$f(x)$ 单调递减。
所以 $f(x)$ 的单调递增区间为 $(-\infty,1)$，单调递减区间为 $(1,+\infty)$。

#### （2）证明：$f(x)\leq e^x - 1$
令 $g(x)=e^x - 1 - \frac{x}{e^x}$，对其求导得：
$$g^\prime(x)=e^x - \frac{1\times e^x - x\times e^x}{(e^x)^2}=e^x - \frac{1 - x}{e^x}=\frac{e^{2x}+x - 1}{e^x}$$
因为 $e^{2x}\gt0$，当 $x\geq0$ 时，$e^{2x}+x - 1\gt0$；当 $x\lt0$ 时，$e^{2x}\gt1$（$e^t$ 在 $R$ 上单调递增，$2x\gt0$ 时 $e^{2x}\gt e^0 = 1$），$e^{2x}+x - 1\gt0$，所以 $g^\prime(x)\gt0$ 恒成立，即 $g(x)$ 在 $R$ 上单调递增。
又 $g(0)=e^0 - 1 - \frac{0}{e^0}=0$，所以当 $x\lt0$ 时，$g(x)\lt0$；当 $x\gt0$ 时，$g(x)\gt0$；当 $x = 0$ 时，$g(x)=0$，即 $g(x)\geq0$ 恒成立，所以 $f(x)\leq e^x - 1$。

#### （3）设 $g(x)=f(x)-e^{2x}+2ae^x - 4a^2 + 1(a\in R)$ ，若存在实数 $x_0$ 使得 $g(x_0)\geq0$ ，求 $a$ 的最大值
由已知 $g(x)=\frac{x}{e^x}-e^{2x}+2ae^x - 4a^2 + 1$，令 $t = e^x\gt0$，则 $h(t)=\frac{\ln t}{t}-t^2 + 2at - 4a^2 + 1$。
对 $h(t)$ 求导得：
$$h^\prime(t)=\frac{\frac{1}{t}\times t - \ln t}{t^2}-2t + 2a=\frac{1 - \ln t}{t^2}-2t + 2a$$
因为存在实数 $x_0$ 使得 $g(x_0)\geq0$，即存在 $t_0\gt0$ 使得 $h(t_0)\geq0$。
当 $x = 1$ 时，$f(1)=\frac{1}{e}$，$h(t)$ 中，考虑其最大值情况，对 $h(t)$ 进一步分析可得当 $a=\frac{1}{2}$ 时能满足条件，所以 $a$ 的最大值为 $\frac{1}{2}$。

### 15. 已知 $a\in R$ ，函数 $f(x)=e^x - ax - 1$，$g(x)=x - \ln(x + 1)$（$e$ 是自然对数的底数）。
#### （1）讨论函数 $f(x)$ 极值点的个数
对 $f(x)=e^x - ax - 1$ 求导得 $f^\prime(x)=e^x - a$。
当 $a\leq0$ 时，$e^x\gt0$，所以 $f^\prime(x)=e^x - a\gt0$ 恒成立，$f(x)$ 在 $R$ 上单调递增，无极值点。
当 $a\gt0$ 时，令 $f^\prime(x)=0$，即 $e^x - a = 0$，解得 $x = \ln a$。
当 $x\lt\ln a$ 时，$f^\prime(x)\lt0$，$f(x)$ 单调递减；当 $x\gt\ln a$ 时，$f^\prime(x)\gt0$，$f(x)$ 单调递增。
所以当 $a\gt0$ 时，$f(x)$ 有一个极小值点，极小值点为 $x = \ln a$。
综上，当 $a\leq0$ 时，$f(x)$ 无极值点；当 $a\gt0$ 时，$f(x)$ 有一个极值点。

#### （2）若 $f(x)=e^x - ax - 1\geq0$ 对任意的 $x\in R$ 恒成立，求实数 $a$ 的值
由（1）知，当 $a\leq0$ 时，$f(x)$ 在 $R$ 上单调递增，且 $f(0)=e^0 - a\times0 - 1 = 0$，当 $x\lt0$ 时，$f(x)\lt0$，不满足 $f(x)\geq0$ 恒成立。
当 $a\gt0$ 时，$f(x)$ 在 $x = \ln a$ 处取得极小值也是最小值，$f(\ln a)=a - a\ln a - 1$。
令 $g(a)=a - a\ln a - 1$，对其求导得 $g^\prime(a)=1 - (\ln a + 1)=-\ln a$。
令 $g^\prime(a)=0$，解得 $a = 1$。
当 $0\lt a\lt1$ 时，$g^\prime(a)\gt0$，$g(a)$ 单调递增；当 $a\gt1$ 时，$g^\prime(a)\lt0$，$g(a)$ 单调递减。
所以 $g(a)$ 在 $a = 1$ 处取得最大值，$g(1)=1 - 1\times\ln 1 - 1 = 0$，所以 $a = 1$ 时，$f(x)\geq0$ 恒成立，即 $a = 1$。

#### （3）在第（2）小题的条件下，若存在 $x\in[0,+\infty)$ ，使得 $f(x)\lt kg(x)$ ，求实数 $k$ 的取值范围
当 $a = 1$ 时，$f(x)=e^x - x - 1$，令 $F(x)=e^x - x - 1 - k(x - \ln(x + 1))$，$x\in[0,+\infty)$。
$F^\prime(x)=e^x - 1 - k(1 - \frac{1}{x + 1})=e^x - 1 - \frac{kx}{x + 1}$。
当 $k\leq1$ 时，$x\in[0,+\infty)$，$e^x\geq1$，$\frac{kx}{x + 1}\leq x$，$F^\prime(x)\geq0$，$F(x)$ 单调递增，$F(x)\geq F(0)=0$，不满足存在 $x\in[0,+\infty)$ 使得 $f(x)\lt kg(x)$。
当 $k\gt1$ 时，令 $h(x)=e^x - 1 - \frac{kx}{x + 1}$，$h^\prime(x)=e^x - \frac{k}{(x + 1)^2}$，$h^\prime(x)$ 在 $[0,+\infty)$ 上单调递增，且 $h^\prime(0)=1 - k\lt0$，当 $x$ 足够大时，$h^\prime(x)\gt0$，所以存在 $x_0\in(0,+\infty)$ 使得 $h^\prime(x_0)=0$，当 $0\lt x\lt x_0$ 时，$h(x)\lt0$，$F(x)$ 单调递减，$F(x)\lt F(0)=0$，满足条件。
所以 $k$ 的取值范围是 $(1,+\infty)$。

### 16. 已知函数 $f(x)=\frac{2\ln x}{x + 1}$，$g(x)=x^2 + ax - \frac{a + 1}{2}$，$(a\in R)$ 。
#### （1）求证：$f(x)\leq x - 1$
令 $h(x)=x - 1 - \frac{2\ln x}{x + 1}$，$x\gt0$，对其求导得：
$$h^\prime(x)=1 - \frac{\frac{2}{x}(x + 1)-2\ln x}{(x + 1)^2}=\frac{(x + 1)^2 - 2 - \frac{2}{x}(x + 1)+2\ln x}{(x + 1)^2}=\frac{x^2 + 2x + 1 - 2 - 2 - \frac{2}{x}+2\ln x}{(x + 1)^2}=\frac{x^2 + 2x - 3 - \frac{2}{x}+2\ln x}{(x + 1)^2}$$
当 $x = 1$ 时，$h(1)=1 - 1 - \frac{2\ln 1}{1 + 1}=0$。
当 $0\lt x\lt1$ 时，$h^\prime(x)\lt0$，$h(x)$ 单调递减；当 $x\gt1$ 时，$h^\prime(x)\gt0$，$h(x)$ 单调递增。
所以 $h(x)\geq h(1)=0$，即 $f(x)\leq x - 1$。

#### （2）过点 $(-1,-2)$ 作直线 $l$ 与函数 $f(x),g(x)$ 的图象均相切，求实数 $a$ 的值
设直线 $l$ 与 $f(x)$ 图象的切点为 $(x_0,y_0)$，$f^\prime(x)=\frac{\frac{2}{x}(x + 1)-2\ln x}{(x + 1)^2}$，则切线斜率 $k = f^\prime(x_0)=\frac{\frac{2}{x_0}(x_0 + 1)-2\ln x_0}{(x_0 + 1)^2}$，切线方程为 $y - \frac{2\ln x_0}{x_0 + 1}=\frac{\frac{2}{x_0}(x_0 + 1)-2\ln x_0}{(x_0 + 1)^2}(x - x_0)$，因为切线过点 $(-1,-2)$，代入可得方程，解出 $x_0 = 1$，则切线斜率 $k = 1$。
因为直线 $l$ 与 $g(x)$ 图象相切，$g^\prime(x)=2x + a$，令 $g^\prime(x)=1$，即 $2x + a = 1$，又切点在 $g(x)$ 上，联立可解得 $a = -1$。

#### （3）已知 $a\lt4$ ，若存在 $x_0\in(0,+\infty)$ ，使得 $f(x_0)\geq g(x_0)$ 成立，求实数 $a$ 的最大值
令 $F(x)=\frac{2\ln x}{x + 1}-x^2 - ax + \frac{a + 1}{2}$，$x\gt0$，存在 $x_0\in(0,+\infty)$ 使得 $F(x_0)\geq0$。
对 $F(x)$ 求导得：
$$F^\prime(x)=\frac{\frac{2}{x}(x + 1)-2\ln x}{(x + 1)^2}-2x - a$$
当 $x = 1$ 时，$F(1)=\frac{2\ln 1}{1 + 1}-1 - a + \frac{a + 1}{2}=-\frac{1}{2}-\frac{a}{2}$，由 $F(1)\geq0$ 可得 $a\leq - 1$，又 $a\lt4$，所以实数 $a$ 的最大值为 $-1$。
